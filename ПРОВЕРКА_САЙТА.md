# 🔍 Проверка статического сайта

## Локальная проверка

### 1. Остановите режим разработки (если запущен)
```bash
# Нажмите Ctrl+C в терминале где запущен npm run dev
```

### 2. Запустите статический сервер
```bash
npm run serve
```

### 3. Откройте браузер
Перейдите по адресу: **http://localhost:8000**

## Если видите стандартную страницу Next.js:

Это означает, что вы смотрите на режим разработки, а не на статическую сборку.

**Решение:**
1. Остановите `npm run dev` (Ctrl+C)
2. Запустите `npm run serve`
3. Откройте http://localhost:8000

## Команды для работы:

```bash
# Сборка статического сайта
npm run build

# Локальный просмотр
npm run serve

# Режим разработки (НЕ для статического просмотра)
npm run dev
```
